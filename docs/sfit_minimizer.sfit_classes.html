class sfit_minimizer.sfit_classes.SFitFunction(data=None, theta=None)¶

Bases: object

Main class for sfit minimization routine. Establishes all the necessary functions for executing the A. Gould algorithm. At a minimum, the user should specify:

EITHER calc_model() OR calc_res()

AND calc_df().

Arguments:
data: np.array with shape (N, 3)

an array of the data to be fitted with columns (x, y, yerr).

theta: list of length M

trial values for the parameters of the function to be fit.

property bmat¶

b matrix from sfit. invert to find c matrix used for stepping and finding sigmas. shape = ( len(theta), len(theta) )

calc_bmat()¶

Calculate the b matrix and store it as self.bmat. b_ij = b_ij + (partial F/partial a_i)(partial F/partial a_j)/error^2

calc_chi2()¶
calc_cmat()¶

Invert the b matrix to find the c matrix.

calc_dchi2()¶

Calculate the gradient of chi2 at each datapoint

calc_df()¶

FUNCTION SPECIFIC: Calculate the derivatives of the fitting function relative to each parameter theta and store as self.df. Should be explicitly defined for a specific class that inherits SFitFunction.

sets self.df =

np.array of shape (M, N) where df[i, k] returns the derivative of the fitting function with respect to parameter i evaluated at point k.

calc_dvec()¶

Calculate the d vector and store it as self.dvec. d_i = d_i + (partial chi2/partial a_i)/2

calc_model()¶

FUNCTION SPECIFIC: Calculate expected values of the model. May be explicitly defined for a specific class that inherits SFitFunction.

sets self.ymod =

np.array of shape (N), where each element k is the value of the model evaluated at data[k, 0].

calc_res()¶

Difference between the data and the model. Either this or calc_model() should be explicitly defined for a specific class that inherits SFitFunction.

sets self.res =

np.array of shape (N), where each element k = model[k] - data[k, 1].

calc_sigmas()¶

Calculate the uncertainty in each parameter theta. sigma_a_i = sqrt(c_ii)

calc_step()¶

Calculate the full proposed step size. Delta a_i = sum_j c_ij * d_j.
